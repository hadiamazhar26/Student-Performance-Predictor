# Student Performance Predictor (Machine Learning Portfolio)

This repository contains my machine learning projects focused on predicting student performance using real-world educational data.

## Projects Included

* **[Student Performance Prediction](./student-performance-prediction)**: A regression project using the UCI Student Performance dataset (Mathematics course) to predict final grades ($G3$, scale 0-20). It compares Linear Regression, Decision Trees, and Random Forests, and evaluates them using MAE, RMSE, and $R^2$. It also explores feature importance to identify key social and academic factors affecting scores.

---

## Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/hadiamazhar26/Student-Performance-Predictor.git
   cd Student-Performance-Predictor
   ```

2. **Install dependencies:**
   ```bash
   pip install numpy pandas matplotlib seaborn scikit-learn jupyter
   ```

3. **Explore the projects:**
   * Open the [student-performance-prediction](./student-performance-prediction) folder.
   * Start Jupyter Notebook:
     ```bash
     jupyter notebook
     ```
   * Open `notebook.ipynb` to view the analysis, visualizations, and model results.
